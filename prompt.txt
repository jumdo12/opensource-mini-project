파이테스트 환경을 구축하고, feat/test 브랜치를 새로 만들어줘.

그 다음에 TDD 사이클로 기능을 구현해줘. 기능은 네가 이 프로젝트에서 필요하다고 판단하는 걸로 선택해서 해줘.

TDD 방식으로 진행하되, 커밋을 두 단계로 나눠서 해줘:
1. RED 커밋: 테스트 코드만 먼저 작성하고 커밋 (아직 구현 안 해서 테스트 실패 상태)
2. GREEN 커밋: 실제 기능을 구현해서 테스트가 통과하도록 만든 후 커밋

구체적인 요구사항:
- pytest와 pytest-flask를 requirements.txt에 추가하고 설치할 것
- tests/ 디렉토리 만들고 테스트 파일 작성할 것
- conftest.py에 공통 픽스처(Flask test client, 데이터 초기화) 설정할 것
- RED 커밋 전에 pytest 실행해서 실제로 실패하는 거 확인할 것
- GREEN 커밋 전에 pytest 실행해서 전부 통과하는 거 확인할 것
- 커밋 메시지에 [RED], [GREEN] 태그 붙여줄 것

구현 완료 후 웹에서 직접 확인할 수 있게 서버도 켜줘.
확인 끝나면 메인 브랜치에 머지하고 푸시까지 해줘.
