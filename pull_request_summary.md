### 해결된 코드 스멜

#### 1. 매직 넘버 → 명명된 상수
**이전:**
```python
if len(name) > 20 or len(message) > 200:
```
**이후:**
```python
MAX_NAME_LENGTH = 20
MAX_MESSAGE_LENGTH = 200
if len(name) > MAX_NAME_LENGTH or len(message) > MAX_MESSAGE_LENGTH:
```

#### 2. 원시 타입 강박 → 데이터 클래스
**이전:**
```python
guestbook_entries.append({
    'name': name,
    'message': message,
    'date': datetime.now().strftime('%Y-%m-%d %H:%M'),
})
```
**이후:**
```python
@dataclass
class GuestbookEntry:
    name: str
    message: str
    date: str

entry = GuestbookEntry.create(name, message)
guestbook_entries.append(entry)
```
